# 57 Consulting Site

Eigenes Webprojekt fuer den persoenlichen Aussenauftritt von Dominik Preiss mit
`57 Consulting` als Label im Hintergrund.

## Ziel

- jetzt: starke Landingpage fuer erste Sichtbarkeit und Outreach
- spaeter: Datenbankgestuetzte Inhalte, Leads, Outreach, Events und interaktive
  Strategy-/Use-Case-Bausteine

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma
- Notion API
- Docker Compose

## Start

1. `.env.example` nach `.env.local` kopieren
2. Abhaengigkeiten installieren
3. Postgres mit Docker Compose starten
4. Prisma Client generieren
5. Entwicklungsserver starten

## Hinweise

- Notion ist vorerst die redaktionelle Source of Truth.
- Die Datenbank ist fuer strukturierte Anwendungsschicht und spaeteren Ausbau da.
- Kontaktdaten sind aktuell Platzhalter.
